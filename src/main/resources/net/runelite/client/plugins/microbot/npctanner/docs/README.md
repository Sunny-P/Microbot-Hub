# NPC Tanner

![preview](assets/card.png)

Tans any hide in the bank.

## Features
- Supports stamina and energy potions.
- Anti Pattern
- Any hide

## Requirements
1. Coins.
2. Hide(s) to tan.

## Setup
1. Install and enable the plugin in your client.
2. Adjust plugin settings to match your preferences.
